# React Digital Clock

Simple React frontend that shows a live digital clock.

## How to run

1. Unzip the project.
2. In the project folder, run:
   ```
   npm install
   npm start
   ```
3. The app will open at http://localhost:3000 and show a live digital clock.

Files included:
- public/index.html
- src/index.js
- src/App.js
- src/Clock.js
- src/index.css
- package.json
